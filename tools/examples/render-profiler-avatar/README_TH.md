# Render และ Profiler Example

คัดลอกโฟลเดอร์นี้ไปไว้ใน `shyne-mods/avatars/` แล้วเลือก Avatar จากคลัง จะเห็น text, item, block, sprite, line และ world-anchored text จากนั้นเปิด `Shyne Settings → Advanced → Avatar Profiler` เพื่อดูค่าใช้จ่ายแบบสด

ตัวอย่างนี้เป็น Creator fixture ภายนอกและไม่ถูกบรรจุใน Mod JAR
