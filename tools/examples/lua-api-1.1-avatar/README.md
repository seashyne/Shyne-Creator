# Shyne Lua API 1.1 Example

ตัวอย่างนี้สาธิต:

- `api: "latest"` และ `requires`
- `shyne.api.require(...)`
- Vector math และ `vector.lerp`
- `events.once(...)`
- `task.every(...)`
- การตรวจ `permissions.has(...)` ก่อนวาด HUD

คัดลอกทั้งโฟลเดอร์ไปไว้ใน `shyne-mods/avatars/` แล้วเปิดจาก Avatar Library
